sobre o git:
Git é o sistema de controle de versão open source mais usado no mundo atualmente!
Ele é usado para controlar o histórico de alterações de arquivos e principalmente de
projetos de desenvolvimento de software.Ele permite mais flexibilidade no fluxo de trabalho,
segurança e desempenho.
Sobre github:
Em suma, o GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão (VCS) chamado Git. Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso.
